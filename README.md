## ❓ Why This Repo?

Memory is the foundation of intelligence — both artificial and human. This repo is a **comprehensive collection** of:

- 📚 Research papers on LLM memory systems
- 🧬 Human brain memory research
- 🧠 Psychological theories of memory
- 💻 Open source memory applications
- 🔬 Neuroscience discoveries
- 🤖 Agentic memory research
- 📖 Books, courses, and videos
- 🛠️ Tools and frameworks

> **Mission:** Create the "Mecca" — the ultimate resource for everyone working on, researching, or interested in memory.

# 🧠 HUMAN MEMORY RESEARCH

## 🧬 Neuroscience

### Key Papers

| Paper | Year | Description |
|-------|------|-------------|
| [Hebbian Theory](https://en.wikipedia.org/wiki/Hebbian_theory) | 1949 | "Cells that fire together, wire together" |
| [Long-term Potentiation](https://www.nature.com/articles/nn.3625) | 2008 | Synaptic plasticity |
| [Hippocampal Memory Formation](https://pubmed.ncbi.nlm.nih.gov/) | 2019 | Role of hippocampus |
| [Engram Cells](https://science.sciencemag.org/content/353/6298/453) | 2016 | Memory engram discovery |
| [The Molecular Biology of Memory](https://pubmed.ncbi.nlm.nih.gov/15617200/) | 2004 | Eric Kandel's Nobel work |

### Key Concepts

- **Synaptic Plasticity** - Brain's ability to change
- **Long-term Potentiation (LTP)** - Synapse strengthening
- **Hippocampus** - Memory formation
- **Prefrontal Cortex** - Working memory
- **Amygdala** - Emotional memory
- **Engram Cells** - Physical memory traces

## 🧪 Psychology

### Classic Papers

| Paper | Year | Author | Description |
|-------|------|--------|-------------|
| [The Magical Number Seven](https://psychology.wikia.org/wiki/The_Magical_Number_Seven,_Plus_or_Minus_Two) | 1956 | George Miller | Working memory capacity |
| [Levels of Processing](https://pubmed.ncbi.nlm.nih.gov/11283312/) | 1972 | Craik & Lockhart | Memory depth theory |
| [Working Memory](https://psycnet.apa.org/doi/10.1037/0033-295X.100.6.628) | 2007 | Baddeley | Working memory model |

### Memory Types

| Type | Description |
|------|-------------|
| **Sensory Memory** | Brief impressions (<1 sec) |
| **Working Memory** | Active manipulation (7±2 items) |
| **Short-term Memory** | Temporary (15-30 sec) |
| **Long-term Memory** | Permanent (unlimited) |
| **Episodic** | Personal experiences |
| **Semantic** | Facts and knowledge |
| **Procedural** | Skills and habits |
