## Key Concepts

### Vector Memory
- Embedding-based storage
- Semantic similarity search
- Approximate nearest neighbors

### Graph Memory
- Entity-relationship modeling
- Belief graphs
- Knowledge graphs

### Hybrid Memory
- Combined vector + graph
- Best of both worlds
- Complex reasoning

### Memetic Vectors (NexusMemory)
- Semantic + Outcome + Temporal + Privacy encoding
- Pattern learning
- Universal domain support
