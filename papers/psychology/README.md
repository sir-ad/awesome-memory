### Memory Processes

| Process | Description | Key Variables |
|---------|-------------|---------------|
| Encoding | Converting experience to storable representation | Attention, depth of processing |
| Storage | Maintaining traces over time | Consolidation, sleep |
| Retrieval | Accessing stored memory | Cue match, context reinstatement |
| Consolidation | Stabilizing labile memory traces | Time, sleep, protein synthesis |
| Reconsolidation | Re-stabilizing after retrieval | Reactivation, prediction error |

### Forgetting Mechanisms

| Mechanism | Description | Key Finding |
|-----------|-------------|-------------|
| Decay | Memory trace fades with time | Ebbinghaus (1885) |
| Interference | Competing memories disrupt recall | Proactive / retroactive interference |
| Retrieval Failure | Cues not available at retrieval | Encoding specificity (Tulving & Thomson) |
| Motivated Forgetting | Active suppression of unwanted memories | Think/No-think paradigm (Anderson et al.) |
| Reconsolidation Failure | Disruption during re-storage after recall | Nader et al. (2000) |

## 📊 Key Theories

### 1. Multi-Store Model (Atkinson & Shiffrin, 1968)

```
SENSORY → SHORT-TERM → LONG-TERM
  Memory     Memory       Memory
    ↓           ↓            ↓
  Seconds   15–30 sec   Lifetime
  (icon/echo) (rehearsal) (consolidation)
```

### 2. Levels of Processing (Craik & Lockhart, 1972)

```
┌────────────────────────────────────────┐
│         LEVELS OF PROCESSING            │
├────────────────────────────────────────┤
│                                        │
│  SHALLOW  →  Structural, phonemic     │
│              (least durable)           │
│                                        │
│  MEDIUM   →  Semantic, meaning        │
│              (moderate durability)     │
│                                        │
│  DEEP     →  Elaborative, context    │
│              (most durable)           │
│                                        │
└────────────────────────────────────────┘
```

### 3. Schema Theory (Bartlett, 1932)

- Memory organized into knowledge schemas (frameworks)
- New information assimilated or accommodated into existing schemas
- Recall is reconstructive, not reproductive — schemas fill in gaps
- Explains memory distortions and false memories

### 4. Dual Coding Theory (Paivio, 1971)

- Information encoded in both verbal and visual channels
- Dual-coded items (words + images) remembered better than single-coded
- Basis for multimedia learning and mnemonic techniques

## 📈 Important Studies

### Serial Position Effect

```
┌─────────────────────────────────────────────┐
│           SERIAL POSITION CURVE              │
│                                              │
│  Recall                                      │
│  (%)                                         │
│  100%  ▄▄                             ▄▄▄▄  │
│   75%  ▄▄▄▄▄                        ▄▄▄▄▄   │
│   50%        ▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄          │
│  ────────────────────────────────────────► │
│   Beginning        Middle          End       │
│   (Primacy)                     (Recency)    │
│                                              │
│  Primacy: transferred to long-term memory    │
│  Recency: still in working memory            │
└─────────────────────────────────────────────┘
```

### von Restorff Effect (Isolation Effect)

- Distinctive items are remembered significantly better than surrounding items
- Basis for educational techniques like highlighting and visual emphasis
- Applies across modalities (visual, auditory, semantic distinctiveness)

### Spacing Effect (Distributed Practice)

- Studying spread over time produces far better long-term retention than massed practice
- Ebbinghaus (1885) discovered; Cepeda et al. (2006) quantified optimal spacing
- Foundational principle behind spaced repetition systems (Anki, SuperMemo, FSRS)

### Testing Effect (Retrieval Practice)

- Retrieving information from memory strengthens it more than re-studying
- Roediger & Karpicke (2006) seminal study: test > restudy for long-term retention
- Explains why active recall is more effective than passive review

## 🔗 Bridge to AI

How psychological memory concepts map to machine learning and AI systems:

| Psychology Concept | AI / ML Analogue | Example |
|-------------------|-----------------|---------|
| Working memory (7±2 chunks) | Context window length | Transformer attention span |
| Long-term potentiation | Gradient descent weight updates | Backpropagation learning |
| Levels of processing | Representation depth | Shallow vs. deep features in CNNs |
| Schema theory | Prior knowledge / pretraining | Foundation model priors |
| Encoding specificity | Retrieval context matching | RAG query-document alignment |
| Spacing effect | Curriculum learning / replay | Experience replay in RL |
| Forgetting curve | Memory decay functions | Time-weighted retrieval in mem0 |
| Reconsolidation | Continual / lifelong learning | Preventing catastrophic forgetting |
| Chunking | Tokenization / subword segmentation | BPE tokenizers |
| Serial position effect | Primacy/recency bias in attention | "Lost in the Middle" (Liu et al., 2023) |
