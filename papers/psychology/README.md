### Memory Processes

| Process | Description |
|---------|-------------|
| Encoding | Converting to storable form |
| Storage | Maintaining over time |
| Retrieval | Accessing stored memory |
| Consolidation | Stabilizing memory |
| Reconsolidation | Re-stabilizing after retrieval |

### Forgetting Mechanisms

| Mechanism | Description |
|-----------|-------------|
| Decay | Memory trace fades |
| Interference | Competition |
| Retrieval Failure | Cues not available |
| Motivated Forgetting | Active suppression |

## 📊 Key Theories

### 1. Multi-Store Model (Atkinson & Shiffrin)

```
SENSORY → SHORT-TERM → LONG-TERM
  Memory     Memory       Memory
    ↓           ↓            ↓
  Seconds   15-30 sec   Lifetime
```

### 2. Levels of Processing

```
┌────────────────────────────────────────┐
│         LEVELS OF PROCESSING            │
├────────────────────────────────────────┤
│                                        │
│  SHALLOW    →  Structural, phonemic    │
│               (brief)                   │
│                                        │
│  MEDIUM     →  Semantic, meaning       │
│               (moderate)               │
│                                        │
│  DEEP       →  Elaborative, context   │
│               (lasting)               │
│                                        │
└────────────────────────────────────────┘
```

### 3. Schema Theory

- Memory organized into schemas
- New info integrated with existing
- Expectations influence recall

## 📈 Important Studies

### Serial Position Effect

```
┌─────────────────────────────────────────────┐
│           SERIAL POSITION CURVE              │
│                                              │
│  █                                          │
│  █  █                                       │
│  █  █  █                                    │
│  █  █  █  █                                │
│  █  █  █  █  █   █                        │
│ ──────────────────────────────────────────►
  Beginning    Middle       End
  (Primacy)              (Recency)
```

### von Restorff Effect

- Distinctive items remembered better
- Isolation effect

### Mood-Congruent Memory

- Happy memories easier to recall when happy
- Sad memories easier when sad
